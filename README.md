# LearnHub

## Project Introduction
Welcome to LearnHub, an online learning platform built with React 19 and Vite. Our mission is to provide a comprehensive and engaging learning experience for users of all ages.

## Folder Structure
```
/src              # Source files
  ├── components  # React components
  ├── assets      # Images and other assets
  ├── pages       # Page components
  ├── hooks       # Custom hooks
  └── utils       # Utility functions
/public           # Public files
  └── index.html  # Main HTML file
```
The source directory contains all the React components and assets used to construct the application, ensuring a modular and maintainable codebase.

## Key Features Overview
- User-friendly interface
- Variety of courses across multiple subjects
- Progress tracking and course completion certificates
- Interactive quizzes and assignments

## Tech Stack
- **Front-end:** React 19
- **Build Tool:** Vite
- **Styling:** CSS-in-JS with styled-components

## Getting Started Instructions
1. Clone the repository: `git clone https://github.com/Rohith-180503/learnhub.git`
2. Navigate to the directory: `cd learnhub`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
